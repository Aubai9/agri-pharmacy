# Agri-Pharmacy Management System 🌿

A comprehensive, web-based Point of Sale (POS) and Inventory Management solution specifically designed for Agricultural Pharmacies. This application works 100% offline using PWA technology and stores data locally for maximum privacy and speed.

## 🚀 Key Features

- **Dynamic Dashboard:** Real-time tracking of daily/monthly sales, total debt in the market, inventory value, and net profit.
- **Smart Inventory:** Specialized product tracking including manufacturer, country of origin, texture (Liquid, Powder, etc.), and package size.
- **Advanced POS:** Fast invoicing system with smart customer search and stock auto-deduction.
- **Debt Management (Cash Book):** Complete ledger for farmers and customers. Track "Cash In/Out" with detailed transaction history and notes.
- **Smart Reports:**
    - Profit & Loss analysis with custom date filters.
    - Low Stock Alerts: Automatic identification of items running out.
    - Export to Excel: Generate professional CSV reports for accounting.
- **PWA Integration:** Installable on Desktop and Mobile. Works perfectly without an internet connection after the first load.
- **Secure Access:** Built-in login system with session persistence to protect financial data.

## 🛠️ Tech Stack

- **Frontend:** HTML5, CSS3 (Modern Responsive UI).
- **Logic:** Vanilla JavaScript (ES6+).
- **Database:** LocalStorage / Browser-based persistent storage.
- **Icons:** FontAwesome 6.4.
- **Offline Engine:** Service Workers (PWA).
